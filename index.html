Tempura Demo

a collection of angular components and tools developed or modified by GrubHub. Some of them are original and others are customizations of Twitter Bootstraps.

Typeahead

Typeahead is a modified version of Angular Bootstraps' typeahead component with some new features including:
  • touch screen support
  • open on focus
  • list refreshing
and removing:
  • hint
  • highlight
The typeahead reload function is exposed to matches so that matches can trigger updates.

Popover

Popover is a lightweight popover solution that does not rely on scope.
